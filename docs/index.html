AI-powered generation

Uses Cursor CLI with GPT-5 by default. Handlebars templates as fallback when CLI is unavailable.

Quick start

npm install -g reposage
reposage generate .

Requires Node.js 18+ and Cursor CLI for AI generation. Use --no-ai for template-based output.

What you get

.cursor/

  • • Rules: baseline, architecture, testing, security, tech stack
  • • Commands: explain-repo, trace-feature, create-test-plan, document-module, review-risk

docs/context/

  • • Repo map, architecture overview, module index
  • • Coding conventions, testing strategy, glossary, known risks
